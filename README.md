# go-pack-static-resource
go打包静态资源和html示例

#### 打包静态html文件所需包：

```shell
go get github.com/jessevdk/go-assets-builder
```

#### 打包静态文件所需包：

```shell
go get github.com/jteeuwen/go-bindata/...
go get github.com/elazarl/go-bindata-assetfs/...
```
